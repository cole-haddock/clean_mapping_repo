This is a map of encampment sweeps in Oakland, CA, from 2021 - December 2025. It is meant to be a tool for journalists and researchers. It was created as part of a larger project 
for Street Spirit News titled Swept Off the Map. 

Swept Off the Map is a seven-month investigation into the scope and scale of encampment sweeps in Berkeley, Richmond and Oakland. 
This map is one of five published pieces that comprise this investigation. The rest of the pieces can be found on the Street Spirit website 
and on the Swept Off the Map website: www.sweptoffthemap.com
